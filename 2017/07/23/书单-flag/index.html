前言

  • 其实啊，早在大一暑假在济南文东集训的时候，深感知识不够用，便写过一个书单，花了从CTF比赛中挣来的第一笔奖金，买了一个小kindle，天真的以为kindle里的亚马逊书库以及自己整理的pdf文件，有了小kindle我就能学个不停了，发现内存不够，又买了三星64g的SD卡，发现果然，界面丑又low，亚马逊书库很多想看的书不仅不免费还死贵。看了没两天竟然开始卡，就放弃了电子读书的大业。
  • 再后来啊，不太好意思和父亲抢他下棋娱乐的iPad，于是用奖学金和杭州G20的工资让我买了第一台属于自己的 Ipad air2，开始用ipad看书，再后来啊，网安纳新和系学生会的好多事让我忙的焦头烂额，终于我发现，我的大学也要即将步入尾声，认识的许多人都已经毕业走向属于他们的岗位，再不看，恐怕也没有机会这么安静无忧无虑的学习了
  • 前几天看了一个东西很有意思：
